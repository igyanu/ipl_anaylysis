# 🏏 IPL Analysis Dashboard (2008–2024)

## 📌 Project Overview

This Power BI dashboard provides a comprehensive analysis of the Indian Premier League (IPL) from 2008 to 2024. The dashboard enables users to explore tournament trends, championship records, team performances, and player statistics through interactive visualizations and dynamic filters.

The analysis is built using IPL match-level and ball-by-ball datasets covering all seasons from 2008 to 2024. The dataset includes match results, player performances, team statistics, season winners, and other key IPL metrics. :contentReference[oaicite:0]{index=0}
